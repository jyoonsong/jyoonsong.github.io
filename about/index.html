Trois, the Simple, Responsive, and User-friendly Jekyll Theme

Cool / Categories  |  2017.10.10

Short Introduction about this blog

This blog is powered by Jekyll and Github Pages. The theme of this blog, Trois is made by JaeYoon Song.

Trois is a Light-weight and Mobile-friendly theme. Trois utilizes:

  • SASS
  • 2-depth Categories
  • Facebook Likes and Comments
  • Search : jekyll-algolia
  • Pagination : jekyll-pagination-v2
  • Infinite Scroll
  • Masonry Layout
  • jekyll-babel
  • jekyll-seo-tag

You can find more information about Trois here:

  • See the Full Demo in Korean
  • View Source at Github
  • If you like it, install it as Gem

You can find more information about JaeYoon Song here:

  • My portfolio website
  • My blog in Korean
  • My Email
